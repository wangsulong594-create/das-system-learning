## 🆘 遇到问题？

### 问题1：找不到部署脚本
**解决**：确保脚本文件在当前目录，使用 `ls`（Mac/Linux）或 `dir`（Windows）查看

### 问题2：Git未安装
**解决**：下载安装Git
- Windows: https://git-scm.com/download/win
- Mac: `brew install git`
- Linux: `sudo apt-get install git`

### 问题3：Token错误
**解决**：
1. 重新访问 https://github.com/settings/tokens
2. 生成新Token
3. 完整复制（不包含前后空格）

### 问题4：推送失败
**解决**：
1. 检查网络连接
2. 确认Token有效
3. 参考脚本中的错误信息
