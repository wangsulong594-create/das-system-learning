# DAS系统硬件详细分析：第四部分 - 光放大器(EDFA)

## 4. 掺铒光纤放大器 (EDFA - Erbium-Doped Fiber Amplifier)

### 4.1 物理原理

EDFA是一种光纤放大器，通过掺铒离子的受激辐射放大光信号。

#### 4.1.1 能级跃迁

掺铒离子有两个主要能级：
- **基态**（$^4I_{15/2}$）
- **激发态**（$^4I_{11/2}$）

**泵浦过程**：
1. 泵浦光（980 nm 或 1480 nm）激发掺铒离子从基态到激发态
2. 激发态的掺铒离子通过受激辐射释放能量，放大信号光
3. 自发辐射产生噪声

#### 4.1.2 功率演变方程

沿光纤传播方向（z），功率随距离变化：

**信号光功率**：
$$\frac{dP_s}{dz} = [\Gamma_s(\sigma_{se} + \sigma_{sa}) N_2 - \Gamma_s \sigma_{sa} (N_1 + N_2) - \alpha_s] P_s$$

**泵浦光功率**：
$$\frac{dP_p}{dz} = -[\Gamma_p(\sigma_{pe} + \sigma_{pa}) N_2 - \Gamma_p \sigma_{pa} (N_1 + N_2) + \alpha_p] P_p$$

其中：
- $P_s, P_p$ = 信号和泵浦光功率
- $N_1, N_2$ = 掺铒离子基态和激发态的浓度
- $\sigma_{se}, \sigma_{sa}, \sigma_{pe}, \sigma_{pa}$ = 受激辐射、受激吸收的截面积
- $\Gamma_s, \Gamma_p$ = 重叠因子（光与光纤模场的耦合）
- $\alpha_s, \alpha_p$ = 背景损耗

### 4.2 增益与噪声指数

#### 4.2.1 小信号增益

对于小功率信号，EDFA的增益可近似为：

$$G = \exp\left(\int_0^L g(z) dz\right)$$

其中 $g(z)$ 是增益系数。

**对数增益（dB）**：
$$G_{dB} = 10 \log_{10}(G)$$

**常见配置**：
- 典型EDFA增益：15-40 dB
- 可配置范围：5-40 dB

**功率增益关系**：
$$P_{out} = P_{in} \times G$$

或
$$P_{out,dB} = P_{in,dB} + G_{dB}$$

#### 4.2.2 噪声指数 (Noise Figure, NF)

EDFA的噪声来自自发辐射。噪声指数定义为：

$$NF = \frac{SNR_{in}}{SNR_{out}}$$

或用dB表示：
$$NF_{dB} = 10 \log_{10}(NF)$$

**最小噪声指数**：
$$NF_{min} = \frac{2n_{sp} - 1}{G/\alpha}$$

其中 $n_{sp}$ 是反演种群因子（通常 $n_{sp} \approx 1.5-2$ 对应最优泵浦）

**典型值**：
- 噪声指数：4-6 dB
- 在高增益（>30 dB）时，可达到 3-4 dB

### 4.3 输出光谱噪声功率

EDFA产生的噪声功率谱密度：

$$P_{ASE} = 2 n_{sp} (G - 1) h \nu_{signal} \Delta\nu_B$$

其中：
- $h$ = Planck常数（$6.626 \times 10^{-34}$ J·s）
- $\nu_{signal}$ = 信号频率（1.55 × 10^14 Hz 对应1550 nm）
- $\Delta\nu_B$ = 放大器带宽
- $n_{sp}$ = 反演种群因子

**在1550 nm处**：
$$P_{ASE} = 2 n_{sp} (G - 1) \times 1.033 \times 10^{-20} \times \Delta\nu_B$$

### 4.4 DAS系统中的应用

在DAS系统的探测光路径中，EDFA用于放大弱的回波信号。

#### 4.4.1 瑞利散射信号的衰减

光在光纤中传播时遭受损耗：
$$P(z) = P_0 \exp(-\alpha z)$$

其中 $\alpha$ 是衰减系数（通常 0.2-0.3 dB/km）。

**示例**：对于10 km光纤，$\alpha = 0.25$ dB/km
$$P(L) = P_0 \times 10^{-0.25 \times 10 / 10} = P_0 \times 10^{-0.25} \approx 0.56 P_0$$

光纤另一端的回波还要再传回：
$$P_{return} = P(L) \times 10^{-(1 \times 10)/10} \approx 0.31 P_0$$

#### 4.4.2 需要的EDFA增益

为了恢复回波功率到可检测水平，需要EDFA增益：

**假设**：
- 探测光功率进入光纤：50 mW
- 往返损耗：约 -5 dB（$\times 0.31$）
- 返回功率：15.5 mW
- 目标：放大到 100 mW（用于干涉测量）

**所需增益**：
$$G = \frac{100 \text{ mW}}{15.5 \text{ mW}} \approx 6.45 \times 1.2 = 8.2$$
$$G_{dB} = 10 \log_{10}(8.2) \approx 9.1 \text{ dB}$$

实际上，通常选择 **15-20 dB的增益**，以获得足够的信噪比余量。

### 4.5 EDFA工作点选择

#### 4.5.1 增益与泵浦功率的关系

EDFA的增益取决于泵浦功率。通常：

**泵浦功率 vs 增益**（典型关系）：
- 泵浦功率 100 mW → 增益 ~5 dB
- 泵浦功率 300 mW → 增益 ~15 dB
- 泵浦功率 500 mW → 增益 ~25 dB
- 泵浦功率 1000 mW → 增益 >35 dB

#### 4.5.2 增益平坦化

宽带EDFA在不同波长的增益不同。使用**增益平坦化滤波器（GFF）**使增益在波长范围内保持均匀。

**要求**：增益平坦性 ±1 dB 在 1530-1565 nm范围内

**应用场景**：DAS系统，需要 15-20 dB 增益

**推荐EDFA模块**：

  - 适合需要高功率输出的应用

### 4.7 EDFA链路设计

**DAS系统中的EDFA放置**：

可以有两种配置：

**配置1：前置放大（Pre-Amplification）**
```
弱回波信号 → EDFA → 混频 → 探测
```
- 优点：提高信噪比
- 缺点：EDFA噪声也被放大

**配置2：后置放大（Post-Amplification）**
```
弱回波信号 → 混频 → 前置放大器 → EDFA → 采样
```
- 优点：先混频处理信号
- 缺点：电子前置放大器噪声较大

**推荐**：配置1（光前置放大）更优，噪声指数更低

### 4.8 光功率预算总结

**完整的探测光路径功率计算**：

1. **激光器输出**：100 mW
2. **50:50分路器**：-3 dB → 50 mW
3. **脉冲调制器(AOM)**：-1.5 dB (70% 效率) → 35 mW
4. **光纤传播往返(10 km)**：-5 dB → 11 mW
5. **EDFA放大**：+15 dB (增益31.6倍) → 348 mW
6. **环形器**：-1 dB → 276 mW

最终进入混频的回波光功率：**276 mW**
