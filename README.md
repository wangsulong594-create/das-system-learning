## 项目简介

这是一个**从硬件到软件、从0到1**的DAS（分布式声波传感）系统完整学习记录。

### DAS系统是什么？

DAS（Distributed Acoustic Sensing）是一种利用光纤作为传感器的新型分布式传感技术，可以同时在数千个点上进行振动和声学信号的测量。

**典型应用**：
- 🌍 地震监测与探矿
- 🏭 管道监测（石油、天然气、水管）
- 🏢 建筑结构监测
- 🌊 海洋地震学
- 💥 爆炸检测与安全监控

### 本项目的目标

1. **深入理解DAS系统的完整工作原理** - 从光学物理、信号处理到数字实现
2. **掌握硬件设计与器件选型** - 学习如何选择和集成各个光学、电子器件
3. **实现完整的信号处理链** - 从原始相位数据到最终的振动信号
4. **分享知识，服务社区** - 记录学习过程，帮助其他学习者

### 第一部分：硬件基础与原理（第1-7章）

#### [01-窄线宽激光器原理与选型](./01-窄线宽激光器原理与选型.md)
- **核心内容**：激光器的基本原理、相干长度、线宽选择
- **关键公式**：
  - 相干长度：$L_c = \frac{c}{\pi \Delta\nu}$
  - 应用场景与激光器类型对应表
- **器件选型**：DFB、DBR、光纤激光器的对比与选择

#### [02-光分路器详细分析](./02-光分路器详细分析.md)
- **核心内容**：分路器的耦合原理、功率分配、对偏振的影响
- **关键指标**：分路比、插入损耗、隔离度
- **器件推荐**：50:50对称分路器 vs 90:10非对称分路器

#### [03-脉冲调制(AOM/EOM)详细分析](./03-脉冲调制(AOM-EOM)详细分析.md)
- **核心内容**：Bragg衍射、电光效应、脉冲宽度与空间分辨率
- **关键公式**：
  - 空间分辨率：$\delta z = \frac{c \tau}{2 n_{eff}}$
  - 脉冲重复频率：$f_{rep} < \frac{c}{2 L n_{eff}}$
- **AOM vs EOM**：应用场景和性能对比

#### [04-光放大器(EDFA)详细分析](./04-光放大器(EDFA)详细分析.md)
- **核心内容**：受激辐射、增益、噪声指数、ASE噪声
- **关键指标**：增益、噪声指数、输出功率
- **系统应用**：增益设计、噪声预算、功率优化

#### [05-环形器与光路隔离](./05-环形器详细分析.md)
- **核心内容**：Faraday旋转、单向隔离、回反光管理
- **关键参数**：隔离度、插入损耗、回反光隔离
- **工程应用**：防护激光器、提高系统稳定性

#### [06-混频与检测详细分析](./06-混频与检测详细分析.md)
- **核心内容**：相干光干涉、IQ混频、平衡检测
- **关键公式**：干涉电流、shot noise、信噪比
- **器件选择**：混合器型号、平衡探测器参数

#### [07-高速ADC采样详细分析](./07-高速ADC采样详细分析(完整版).md)
- **核心内容**：采样定理、量化、混叠、ADC关键参数
- **关键指标**：采样率、分辨率、ENOB、DNL/INL

### 第二部分：数字信号处理（第8-11章）

#### [08-数字IQ解调详细分析](./08-数字IQ解调详细分析.md)

## 🎯 核心知识点速览

### 关键公式汇总

#### 光学基础
```
相干长度：      Lc = c / (π·Δν)           [km]
空间分辨率：    δz = c·τ / (2·neff)       [m]
应变灵敏度：    dφ/dε = 2π·pe·L/λ         [rad/strain]
```

#### 参数关系
```
采样率要求：    fs > 2(fvib,max + Δf)              [MHz]
脉冲频率上限：  frep < c / (2·L·neff)              [kHz]
信噪比：        SNR = (2π·SNRdB/20)^2              [倍数]
噪声放大：      Amplification = (π·fs)^2           [倍数]
```

#### 单位转换
```
相位→应变：     ε [με] = Δφ [rad] / (2π) × 476
应变→速度：     v [m/s] = ceff × ε_dot [1/s]
速度→加速度：   a [m/s²] = dv/dt [m/s²]
```
