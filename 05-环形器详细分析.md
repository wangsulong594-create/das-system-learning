# DAS系统硬件详细分析：第五部分 - 环形器与光路隔离

## 5. 光学环形器 (Optical Circulator)

### 5.1 物理原理

光学环形器是一个三/四端口光学器件，具有单向传输特性。

#### 5.1.1 工作原理

环形器内部使用Faraday旋转器（FR）和偏振分束器（PBS）实现单向隔离：

```
输入端(Port 1)
    ↓
 ┌────┐
 │PBS │ 分离S和P偏振
 └─┬──┘
   │
   ↓
 ┌──────────┐
 │Faraday R │ 旋转45°
 │(FR)      │
 └─┬────────┘
   │
   ↓
 ┌────┐
 │PBS │ 合并偏振
 └─┬──┘
   │
   ├→ 输出端(Port 2)
   └→ 输出端(Port 3)
```

#### 5.1.2 Faraday旋转原理

Faraday旋转角与磁场强度关系：

$$\theta_F = V \cdot B$$

其中：
- $\theta_F$ = Faraday旋转角
- $V$ = Verdet常数（取决于材料，如玻璃 V ≈ 3-10 rad/(T·m)）
- $B$ = 磁感应强度

**在环形器中**，通常使用永久磁铁或电磁铁产生恒定磁场，使光旋转 45°。

### 5.2 环形器的传输特性

理想的三端口环形器的传输矩阵：

$$T = \begin{pmatrix} 0 & 0 & 0 \\ 1 & 0 & 0 \\ 0 & 1 & 0 \end{pmatrix}$$

**含义**：
- Port 1 输入 → Port 2 输出（正向）
- Port 2 输入 → Port 3 输出
- Port 3 输入 → 无输出（被隔离）
- Port 1 → Port 3 的隔离度很高

#### 5.2.1 关键参数

**隔离度 (Isolation)**：
$$I = -10 \log_{10}\left(\frac{P_{transmitted}}{P_{incident}}\right) \text{ [dB]}$$

通常 $I > 40$ dB，意味着相反方向的功率被衰减超过 10000 倍。

**插入损耗 (Insertion Loss)**：
$$IL = -10 \log_{10}\left(\frac{P_{out}}{P_{in}}\right) \text{ [dB]}$$

通常 $IL < 1$ dB

**回反光隔离 (Return Loss)**：
$$RL = -10 \log_{10}\left(\frac{P_{reflected}}{P_{incident}}\right) \text{ [dB]}$$

通常 $RL > 50$ dB

### 5.3 DAS系统中的应用

在DAS系统中，环形器的作用是：
1. **隔离出射光和回波光** - 将探测光发送入光纤，同时接收回波
2. **防止反射光回流** - 保护激光器不被损伤

#### 5.3.1 DAS系统的光路配置

```
激光器(100mW, 1550nm)
    ↓
光分路器(50:50)
 ├→ 探测光路
 │   ├ 脉冲调制(AOM)
 │   ├ 功率放大(EDFA)
 │   └→ 环形器[Port 1]
 │       │
 │       ├[Port 2]→ 光纤传感链路(10km)
 │       │                ↓
 │       │          瑞利散射回波
 │       │                ↓
 │       └[Port 3]← 回波接收
 │           ↓
 │        混频检测
 │
 └→ 本振光路
    └ 直接进入90°光学混合器
```

#### 5.3.2 功率流向分析

**正向传输（激光 → 光纤）**：
$$P_{Port1→2} = P_{EDFA} \times (1 - L_{IL})$$

**反向隔离（防止回波泄漏回激光器）**：
$$P_{Port3→1} = P_{reflection} \times 10^{-I/10}$$

### 5.6 反射光管理

#### 5.6.1 法向反射

光纤端面的Fresnel反射系数：

$$R = \left(\frac{n_1 - n_2}{n_1 + n_2}\right)^2$$

对于光纤(n≈1.468)与空气(n=1)的界面：
$$R = \left(\frac{1.468 - 1}{1.468 + 1}\right)^2 \approx 0.034 = 3.4\%$$

**反射光功率**：
$$P_{reflect} = P_{incident} \times 0.034$$

例如，300 mW的光入射到光纤：
$$P_{reflect} = 300 \times 0.034 = 10.2 \text{ mW}$$

#### 5.6.2 反射光隔离

这反射光被环形器的隔离度进一步衰减：
$$P_{leakage} = P_{reflect} \times 10^{-I/10} = 10.2 \times 10^{-40/10}$$
$$= 10.2 \times 10^{-4} = 1.02 \text{ μW}$$

这个泄漏功率非常小，对激光器和系统的影响可以忽略。

#### 5.6.3 防止法线反射的措施

即使有如此高的隔离度，还可以进一步采取措施：

**使用APC（Angled Physical Contact）连接器**：
- 光纤端面与连接器轴线成8°角
- 反射光不沿原路返回
- 反射光进入光纤缓冲层，不再光学相干传播

**使用光隔离器**（额外安全措施）：
- 在激光器输出端增加光隔离器
- 额外隔离度 >30 dB

### 5.7 偏振模色散 (PMD) 与环形器

在长距离DAS系统中，光纤中会产生偏振模色散。

**推荐**：使用**偏振保持（PM）环形器**，确保：
1. 环形器内部偏振状态稳定
2. 与偏振保持光纤匹配
3. 减少光路中的偏振失配
