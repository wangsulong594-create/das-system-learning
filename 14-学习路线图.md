# DAS系统学习路线图

## 学习阶段与目标

### 第一阶段：基础物理（1-2周）

**目标**：理解DAS系统的基本物理原理

#### 第1周：光学基础
- [ ] **相干光干涉**
  - 学习资源：《光学》(沈泽昂) 第4章
  - 重点：干涉条件、相位、可见度
  - 实践：计算机仿真（Python或MATLAB）

- [ ] **光纤传播**
  - 学习资源：《光纤通信原理》(陈家璨) 第2-3章
  - 重点：模式、偏振、色散、衰减
  - 练习：光纤参数计算

#### 第2周：Rayleigh散射与应变光学
- [ ] **瑞利散射**
  - 原理：光纤晶体不均匀性引起的弹性散射
  - 散射强度：$\propto 1/\lambda^4$

- [ ] **应变光学效应**
  - Pockels效应：$\Delta n = n^3 r E / 2$
  - 光弹效应：$\Delta n \propto \varepsilon$（应变）
  - 应变灵敏度计算

**本阶段成果**：
- 理解相干光混频原理
- 掌握相位与应变的关系
- 能够推导基本公式

---

### 第二阶段：硬件原理（2-3周）

**目标**：理解DAS硬件各个环节的工作原理

#### 第3周：光路设计
- [ ] **窄线宽激光器**
  - DFB/DBR/光纤激光器对比
  - 相干长度计算与应用

- [ ] **光学混频器**
  - Faraday隔离子原理
  - 偏振分束器设计
  - 90°混合器的数学推导

- [ ] **环形器与隔离**
  - 单向传输原理
  - 反射光隔离

#### 第4周：检测与采样
- [ ] **混频与检测**
  - IQ解调原理
  - 平衡探测的优势
  - shot noise与thermal noise

- [ ] **模数转换**
  - Nyquist采样定理
  - ADC关键参数（ENOB、DNL、INL）
  - 混叠与反混叠滤波

#### 第5周：系统指标与设计
- [ ] **参数权衡**
  - 空间分辨率 vs 信噪比
  - 采样率 vs 数据速率
  - 脉冲重复频率 vs 测程

- [ ] **噪声预算**
  - 各级噪声来源
  - 总体系统SNR计算

**本阶段成果**：
- 能够设计基本的DAS光路
- 理解各部件的工作原理
- 能够进行初步的参数优化

---

### 第三阶段：数字信号处理（2-3周）

**目标**：掌握DAS数据处理的各个环节

#### 第6周：IQ解调与相位提取
- [ ] **数字下变频 (DDC)**
  - DDS原理
  - 混频与滤波
  - FPGA实现框架

- [ ] **相位提取**
  - atan2函数与四象限处理
  - 相位噪声分析
  - 相位估计的 Cramér-Rao界

#### 第7周：相位展开与滤波
- [ ] **相位展开算法**
  - 一维路径跟踪法
  - 二维质量引导法
  - 噪声稳健性

- [ ] **自适应滤波**
  - 卡尔曼滤波原理
  - 自适应LMS算法
  - 应用于实时处理

#### 第8周：时空差分与振动恢复
- [ ] **Gauge Length差分**
  - 空间分辨率优化
  - 多尺度信息提取

- [ ] **时间微分**
  - 一阶导数（速度）
  - 二阶导数（加速度）
  - 噪声放大与防护

**本阶段成果**：
- 掌握DAS数据处理流程
- 能够编程实现基本算法
- 理解各处理步骤的目的

---

### 第四阶段：实践与优化（3-4周）

**目标**：通过编程和仿真，验证理论，优化设计

#### 第9周：MATLAB/Python仿真
- [ ] **建立完整的DAS处理链**
  ```
  原始相位 → IQ解调 → 相位提取 → 相位展开 
  → Gauge差分 → 时间微分 → 输出振动信号
  ```

- [ ] **仿真测试**
  - 纯信号仿真：验证理论准确性
  - 加噪声仿真：验证抗噪性能
  - 极限情况测试

#### 第10周：FPGA原型设计
- [ ] **学习Verilog/VHDL**
  - 基本语法和设计流程
  - 时序设计与SoC概念

- [ ] **实现关键模块**
  - DDC（数字下变频）
  - FIR/IIR滤波器
  - atan2 (CORDIC算法)

#### 第11周：硬件集成与测试
- [ ] **搭建实验装置**
  - 集成光路（如果有条件）
  - 或使用模拟数据进行完整测试

- [ ] **性能评估**
  - 测量系统SNR
  - 验证空间和时间分辨率
  - 检测动态范围

#### 第12周：改进与优化
- [ ] **问题诊断与解决**
  - 性能瓶颈识别
  - 参数调优

- [ ] **设计文档编写**
  - 系统设计说明书
  - 测试报告

**本阶段成果**：
- 完整的DAS处理系统实现
- 性能验证与优化报告
- 可用于后续改进的基础

## 实验项目建议

### 实验1：干涉仪特性测量
**时间**：1周
**目标**：理解干涉原理
**内容**：
- 构建简单的Mach-Zehnder干涉仪
- 测量干涉条纹
- 验证相位与路径差的关系

### 实验2：相位提取与展开
**时间**：2周
**目标**：掌握数据处理
**内容**：
- 用仿真生成包裹相位数据
- 实现展开算法
- 评估不同算法的性能

### 实验3：单通道DAS原型
**时间**：3-4周
**目标**：验证完整的DAS流程
**内容**：
- 搭建单个光纤传感点
- 实现完整的信号处理链
- 测量已知频率的振动

### 实验4：多通道系统
**时间**：4-6周
**目标**：扩展到实用系统
**内容**：
- 集成多个传感点
- 实现实时数据处理
- 性能优化与测试
