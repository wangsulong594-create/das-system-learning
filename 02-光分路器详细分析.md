# DAS系统硬件详细分析：第二部分 - 光分路器

## 2. 光分路器 (Optical Splitter/Coupler)

### 2.1 物理原理

光分路器将激光光束分为两部分：
1. **探测光** → 进入光纤传感链路
2. **本振光** → 用于后续混频（90°光学混合器）

### 2.2 光分路器的传输方程

对于1×2分路器（将光分为两路）：

**功率分配**：
$$P_1 + P_2 = P_{in} \times \eta$$

其中：
- $P_1$ = 第一输出端口功率
- $P_2$ = 第二输出端口功率
- $P_{in}$ = 输入光功率
- $\eta$ = 传输效率 (通常 0.95-0.99)

**分路比（Splitting Ratio）**：
$$r = \frac{P_1}{P_1 + P_2}$$

对于50:50分路器：$r = 0.5$（各路50%功率）

### 2.3 耦合系数与功率分配

**偶模与奇模分析**：

设输入光场为 $E_{in}$，分路器的传输矩阵为：

$$\begin{pmatrix} E_{out,1} \\ E_{out,2} \end{pmatrix} = \begin{pmatrix} \sqrt{1-\kappa} & j\sqrt{\kappa} \\ j\sqrt{\kappa} & \sqrt{1-\kappa} \end{pmatrix} \begin{pmatrix} E_{in,1} \\ E_{in,2} \end{pmatrix}$$

其中 $\kappa$ 是耦合系数：
- 50:50分路器：$\kappa = 0.5$
- 90:10分路器：$\kappa = 0.1$ 或 $0.9$

**输出功率**：
$$P_1 = |E_{out,1}|^2 = (1-\kappa)P_{in}$$
$$P_2 = |E_{out,2}|^2 = \kappa P_{in}$$

### 2.4 DAS系统中的分路器选择

在DAS系统中，探测光和本振光的功率比至关重要。

**信号功率需求分析**：

设混频后的电流为：
$$I \propto \sqrt{P_{signal} \times P_{LO}}$$

为了获得最大信噪比（Shot Noise Limited），需要：
$$P_{LO} >> P_{signal}$$

**推荐配置**：
- 若激光器功率 $P_0 = 100$ mW
- 使用50:50分路器：各路50 mW
- 本振光（LO）：50 mW（用于混频）
- 探测光：50 mW（进入光纤传感链路，经过放大后增强）

或者使用不对称分路器：
- 90:10分路器：90% 用于探测光（90 mW），10% 用于本振光（10 mW）
- 然后用EDFA放大本振光至所需功率

### 2.5 器件选型参数

| 参数 | 说明 | 选择建议 |
|-----|------|---------|
| 分路比 | 功率分配比例 | 50:50 或 90:10 |
| 工作波长 | 通常1550 nm | 1550 nm ±100 nm |
| 插入损耗 | 分路器本身损耗 | <0.5 dB |
| 隔离度 | 端口间隔离程度 | >30 dB |
| 偏振保持 | 是否保持偏振 | 推荐PM（偏振保持）|
| 频带宽度 | 工作频率范围 | >10 GHz |
| 连接方式 | FC/APC, SC/APC 等 | 推荐APC角度连接（低反射）|

### 2.6 器件选型示例

**应用场景**：DAS系统，光纤长10 km，激光器功率100 mW

**选择方案1：50:50对称分路器**
- 推荐型号：**Thorlabs TW1550R5A1**
  - 插入损耗：3.0 dB（实际分路效率：50%)
  - 工作波长：1550 nm
  - 连接：APC角度连接
  - 价格：中等

**选择方案2：90:10不对称分路器**
- 推荐型号：**Corning 98311-51-14.5**
  - 90%功率用于传感
  - 10%功率用于本振参考
  - 插入损耗：<0.5 dB

### 2.7 光功率预算更新

输入激光器功率：$P_0 = 100$ mW

**探测光路径**：
$$P_{probe} = P_0 \times 0.5 \times 0.5 \times 1000 \times 0.79 = 19.75 \text{ W}$$
（50:50分路 × 调制器 × EDFA增益 × 环形器）

**本振光路径**（假设在分路器后单独放大）：
$$P_{LO} = P_0 \times 0.5 = 50 \text{ mW}$$
（或经过额外的EDFA放大到几百mW）

## 光分路器实现示意

```
激光器(100mW)
    │
    ↓
┌─────────────┐
│ 50:50分路器  │ (插入损耗 3dB)
└─────────────┘
   │         │
50mW        50mW
   │         │
   ↓         ↓
 探测光    本振光(LO)
   │         │
   ↓         └──→ 90°光学混合器（混频）
  脉冲调制
   │
   ↓
  EDFA放大
   │
   ↓
  环形器
   │
   ↓
光纤传感链路
```
